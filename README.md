# FIFA World Cup 2026 — Knockout Tracker

Interactive world map tracking every team at the 2026 FIFA World Cup: quarterfinalists in yellow, eliminated nations in red, live matches pulsing orange.

**Live demo:** deployed on Vercel

Built with D3.js + TopoJSON. Single static `index.html` — no build step, no dependencies to install.
